「主張收容所公開透明」　柯文哲再嗆陳時中：先跟我講什麼不能講 

: 武漢肺炎疫情延燒，滯留武漢的台灣人預定今晚包機回台，針對陽明山隔離收容所風波，
: 台北市長柯文哲3日再嗆衛福部長，柯文哲表示自己主張收容所應公開透明，他要陳時中
: 講清楚，什麼可以講、什麼不能講。
: 柯文哲3日上午接受媒體聯訪表示，中央政府要把大陸的台胞撤回來，要標準檢疫14天，
: 全台灣有規劃五個收容地點，台北的收容地點需要北市府支援醫療人力、警衛、後勤補給
: ，所以他看過資料。
: 他認為，「幾百人要在一個收容所，不可能說台灣的家屬不知道，周邊居民不知道，甚至
: 我們去支援的人都不知道，這不可能嘛」，這麼龐大的一個計畫。
: 柯文哲強調，「公開透明就是說，與其在那裡遮躲掩藏，不如就是...我也沒有去講太多
: ，大概就這樣，我沒有特別爆料幹什麼，有什麼惡意，that’s all」、「如果陳時中要
: 講說要什麼罰不罰，那你就先跟我講，什麼是我可以講的，什麼是我不能講的，講清楚」
: 。
: 對於外界有人認為柯文哲失言，柯文哲說，「我只是記者問我隨意回答，但是我不認為這
: 是一個機密」，一定大家都知道的。對於應由指揮中心統一說明，柯文哲回稱，「所以你
: 要跟我講，什麼話我是可以講，什麼話不能講，我是遵守SOP的，不然衛福部給我們台北
: 市公文，上面就給我註明密件」。
: 柯文哲強調，公開透明可以減少很多不必要的爭執。至於引起恐慌，柯表示「地點要選好
: ，然後該做的防備還是要做」。
: 至於是否呼籲中央公布另外四處地點，柯文哲表示，他還是主張寧可事先講，不要人都進
: 去了，那隔一天才被周邊居民發現，「阿不是更慘」。柯說，「衛福部要到最後一秒，人
: 都進去了才要通知旁邊的居民嗎？不可能嘛，到時候抗爭更大」。
